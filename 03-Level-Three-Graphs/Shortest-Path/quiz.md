

## Shortest Path Quiz 
Questions

Q1. What is the main goal of the shortest path problem?
A) Maximize the distance between two vertices
B) Minimize the distance or cost between two vertices
C) Count the number of vertices
D) Detect cycles

Q2. BFS is suitable for which type of graph?
A) Weighted graph
B) Unweighted graph
C) Graph with negative weights
D) Directed graph only

Q3. Dijkstra’s algorithm cannot handle:
A) Positive weights
B) Unweighted graphs
C) Negative edge weights
D) Trees

Q4. Which algorithm can detect negative cycles?
A) BFS
B) Dijkstra
C) Bellman-Ford
D) Floyd-Warshall

Q5. Floyd-Warshall is primarily used for:
A) Single-source shortest path
B) Single-pair shortest path
C) All-pairs shortest path
D) Detecting cycles

Q6. Johnson’s algorithm is most efficient for:
A) Dense graphs
B) Sparse graphs
C) Unweighted graphs
D) Trees

Q7. What is “relaxation” in shortest path algorithms?
A) Skipping a vertex
B) Updating the shortest distance to a vertex if a better path is found
C) Removing edges
D) Sorting vertices

Q8. BFS finds the shortest path based on:
A) Edge weights
B) Number of edges
C) Vertex degrees
D) Minimum cost

Q9. Bellman-Ford time complexity is:
A) O(V + E)
B) O(V²)
C) O(V * E)
D) O(E log V)

Q10. Dijkstra’s time complexity using adjacency matrix is:
A) O(V + E)
B) O(V²)
C) O(V * E)
D) O(E log V)

Q11. In Floyd-Warshall, the update formula is:
A) dist[i][j] = max(dist[i][j], dist[i][k] + dist[k][j])
B) dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
C) dist[i][j] = dist[i][k] - dist[k][j]
D) dist[i][j] = dist[i][k] * dist[k][j]

Q12. Which algorithm can find the shortest path between two specific vertices efficiently?
A) BFS
B) Dijkstra
C) Bellman-Ford
D) All of the above

Q13. Negative cycles cause the shortest path distance to be:
A) Zero
B) Infinity
C) Undefined
D) Always positive

Q14. Which algorithm uses a priority queue to select the next vertex?
A) BFS
B) Dijkstra
C) Bellman-Ford
D) Floyd-Warshall

Q15. BFS time complexity is:
A) O(V)
B) O(E)
C) O(V + E)
D) O(V * E)

Q16. Bellman-Ford relaxes edges how many times?
A) V
B) V-1
C) E
D) E-1

Q17. Dijkstra’s algorithm always finds the correct shortest path if:
A) There are negative edges
B) All edges are non-negative
C) Graph is disconnected
D) Graph is directed

Q18. Johnson’s algorithm combines which two algorithms?
A) BFS + DFS
B) Bellman-Ford + Dijkstra
C) Floyd-Warshall + Dijkstra
D) BFS + Dijkstra

A) Dijkstra
B) Bellman-Ford
C) BFS
D) Floyd-Warshall
