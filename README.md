<h1 style="color:#1f2933; font-size:2.4rem; margin-bottom:10px;">Data Structures Roadmap</h1>

<p>This repository is a <strong>structured and learning-focused roadmap</strong> for studying <strong>Data Structures</strong> using <strong>C++</strong>.</p>

<p>The goal of this project is to move beyond memorization and focus on:</p>

<ul>
<li>Understanding <strong>why</strong> each data structure exists</li>
<li>Knowing <strong>when</strong> and <strong>why</strong> to use it</li>
<li>Analyzing time and space complexity</li>
<li>Writing clean implementations from scratch</li>
<li>Recommended applications and use cases after completing each topic</li>
</ul>

<div style="background:#eef2ff; border-left:5px solid #6366f1; padding:15px; border-radius:6px; margin-top:20px;"> 
This roadmap is designed for <strong>deep understanding</strong>, not speed. Each topic is studied carefully and incrementally.
</div>

<h2 style="margin-top:40px; border-bottom:2px solid #e5e7eb; padding-bottom:5px;">Learning Workflow</h2>

<div style="background:#fff; padding:20px; border-radius:8px; margin-top:20px; box-shadow: 0 2px 8px rgba(0,0,0,0.05);">
<p>Each topic in this repository follows a consistent learning flow:</p>
<ol>
<li><strong>Summary</strong> – Clear and concise explanation with a full PDF reference for the topic</li>
<li><strong>Quiz</strong> – Short self-test to validate understanding for the topic</li>
<li><strong>Implementation</strong> – Code written from scratch</li>
<li><strong>Applications</strong> – Suggested mini projects and practical applications to reinforce understanding</li>
</ol>
<p><strong>Note:</strong> Level 0 is theory-only (no code).</p>
</div>
